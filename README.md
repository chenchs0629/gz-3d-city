# gz-3d-city

一个基于 Three.js 的 3D 城市漫游系统，支持瓦片动态加载与建筑生长动画。

## 功能简介
- **城市漫游**：相机可在城市区域自由切换视角，支持 7 档鸟瞰到街景的平滑过渡。
- **瓦片加载**：根据相机位置动态加载/卸载城市瓦片数据，提升性能。
- **建筑动画**：建筑加载时采用"拔地而起"生长动画，配合路网淡入效果，视觉流畅。
- **天空盒**：支持 HDRI 天空背景，提升真实感。
- **全览模式**：向上滚轮进入宏观视角，展示城市点云热力图，聚光灯聚焦显示建筑群。
- **建筑信息交互**：鼠标悬浮在建筑上实时显示高度与功能类别（商业区、住宅区、公共服务等）。
- **动态阴影**：支持实时阴影计算，太阳光随相机目标移动，确保视野中心始终被正确照亮。

## 快速启动
1. 安装依赖：
   ```bash
   npm install
   ```
2. 启动开发服务器：
   ```bash
   npm run dev
   ```
3. 浏览器访问：
   ```
   http://localhost:5173/ 或 5174
   ```

## 主要技术点
- Three.js 场景、相机、灯光（DirectionalLight + HemisphereLight）、天空盒
- MapControls 控制器自定义视角切换和平滑过渡
- 城市瓦片数据动态加载与卸载（支持 VISIBLE_RADIUS 配置）
- 建筑生长动画（scale.y 动态变化）+ 路网淡入动画
- Raycaster 射线检测实现鼠标悬浮交互
- 实时阴影计算（PCFSoftShadowMap）
- 聚光灯（SpotLight）实现全览模式圆形高亮
- 宏观点云渲染（PointsMaterial + AdditiveBlending）
- ArcGIS 动态底图加载（Web Mercator 投影转换）

## 数据说明
- 城市瓦片数据存放于 `public/data/tiles/`，每个瓦片为 GeoJSON 格式，包含建筑多边形及以下属性：
  - `Height`：建筑高度（米）
  - `dominant_c`：建筑功能类别（1=商业区、2=住宅区、3=公共服务、4=科技与工业、5=教育文化）
- 道路网数据存放于 `public/data/roadnet/`，采用 GeoJSON Polygon 简化表示。
- 宏观视角点云数据存放于 `public/data/macro_points.json`，为简化格式：`[x, y, type, x, y, type, ...]`。
- 天空盒贴图建议使用 HDRI 或高质量 JPG/PNG，放于 `public/skybox/`。

## 交互说明
- **鼠标左键拖动**：平移视角
- **鼠标右键拖动**：旋转视角
- **鼠标滚轮上滑**：切换到下一档位视角（更贴近地面）
- **鼠标滚轮下滑**：切换到上一档位视角（升高视角）；在最高档位继续下滑进入全览模式
- **鼠标悬浮建筑**：显示该建筑的高度（Height）和功能类别（商业区、住宅区、公共服务、科技与工业、教育文化）成员信息
